# India's Unicorn Revolution 
*By Anya Goel (S4914371)*

An interactive website telling the story of India’s rapid rise as a global startup powerhouse, featuring dynamic data visualisations.


## Installation

No setup required. Simply open the `index.html` file in any modern web browser on a desktop or laptop.


## Usage

Scroll through the page to explore the key sections:

- **Unicorn Growth Race**  
  An animated bar chart race illustrating India’s surge in unicorn startups over time.

- **Industry Connections**  
  Sankey diagrams showing the flow of industries and valuations for Indian and US unicorn startups.


## Files

- `index.html` — Main HTML file containing page structure and embedded charts  
- `style.css` — CSS file defining layout, typography, colours, and animations  
- `sankey.js` — JavaScript file implementing sankey diagram functionality.
- `script.js`: JavaScript file implementing the popup feature.
